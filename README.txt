Manage Website or APP form submissions at one place i.e. RAO Form Builder.

== Description ==
RAO Forms is an omnichannel multi form submission that helps you to dump and manage submission data from your Wordpress website form plugins to RAO Forms. Easily collect leads data from multiple channels and export to csv, pdf, or google sheet to integrate with your CRM.

== Integrated with Renowed Form Plugins ==

== Do you already have contact forms on your wordpress website and having tough time in analyzing submissions?.==

RAO Forms plugin provides a feature to save all the submissions from different forms at one place. It is a one time configuration process. After activating the plugin , please visit the RAO Forms Settings Page. Screenshots are added at the bottom for better understanding.

User submissions will be sent to RAO Forms Submissions whenever visitor will submit the form data from the plugins mentioned below.

*  **[Contact Form 7](https://wordpress.org/plugins/contact-form-7/)**

*  **[WPForms](https://wordpress.org/plugins/wpforms-lite/)**

*  **[Ninja Forms](https://wordpress.org/plugins/ninja-forms/)**

== Embed RAO Forms into WordPress website ==
Embed any type of form into your Wordpress website without adding a server cost to a wordpress website.

**Steps to Embed Rao Forms**

1. Open [Rao Forms App](https://app.raoforms.com/)
2. Login to Rao Forms
3. Create Form with Predefined/Custom Template and add fields using Form Builder
4. Go to Settings tab => Live Form Settings Steps
5. Copy the Embed Code <iframe src="https://live.raoforms.com/live/form_id" width="400" height="800" style="border: 0"></iframe>
